Project Brief: Machine Learning for Quantum State Tomography (ML-QST)

October 2025

1 Project Overview

Quantum State Tomography (QST) is the essential process of reconstructing the density matrix ρ of an unknown quantum state from experimental measurements. While fundamental, conventional QST suffers from the curse of dimensionality, requiring an exponentially growing number of measurements and resources as the number of qubits N increases.

This project focuses on using modern Machine Learning techniques (deep learning and neuromorphic computing) to achieve highly efficient, scalable and real-time Quantum State Tomography. The goal is to move beyond conventional methods and enable state certification on NISQ devices and future fault-tolerant quantum computers.

2 Key Applications and Impact

The developed ML-QST methods will have immediate applications across the quantum ecosystem

  • Quantum Algorithm Verification efficiently certifying circuit outputs
  • Qubit Characterisation rapid measurement of purity, entanglement and fidelity
  • Quantum Sensing precise parameter estimation via tiny changes in ρ
  • Quantum Error Suppression real-time fidelity monitoring during computation
  • Neural Quantum Interface compact classical representation of quantum states

3 Technical Focus Tracks

3.1 Track 1: Neural Shadow Tomography (NSQST)

Implement and optimise Neural-Shadow QST using Classical Shadows combined with neural networks. Deliverable: Python/Julia package capable of high-fidelity tomography of up to 20 qubits with very few measurements.

3.2 Track 2: FPGA Accelerated Tomography

Quantise a trained neural model and implement it on FPGA to achieve sub-millisecond inference on streaming measurement data. Deliverable: low-latency hardware core.

3.3 Track 3: Neuromorphic SNN for QST

Develop a Spiking Neural Network that maps measurement outcomes to state parameters, exploiting event-driven computation. Deliverable: prototype showing clear energy advantage over standard DNNs.
